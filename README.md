## 🧠 Project Overview
**StudentHub** is a full-stack web application designed to manage student data efficiently.  
Built with **Django**, it provides a clean interface and powerful backend to handle student records.

---

## 🔥 Key Features
- 👨‍🎓 Add / Update / Delete Students (CRUD)
- 🔐 Secure Login System
- 🔍 Search Students Instantly
- 📄 View Detailed Profiles
- 🎨 Clean & Responsive UI
- ⚡ Fast Performance
